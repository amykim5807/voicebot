Running the web app:
- Install node/npm
- Install dependencies
- Setup local server (any port)
- Navigate to data_api folder and run 'node server.js'
- Open webpage (localhost:port/index.html)

File Structure:
- HTML pages(index,chat)
- Scripts Folder - scripts(home,chat,chatbox)
- CSS Folder - style.css
- data_api -- Database Query API
    - Controllers
    - Routes
    - server.js
